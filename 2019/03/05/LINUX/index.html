linux常用命令汇总

Linux常用命令

1 目录操作

cd ./a　切换到当前目录的a文件夹
cd .. 　切换到上一层目录
cd /　　切换到系统根目录
cd ~　　切换到用户主目录
cd -　　切换到上一个所在目录

pwd　　显示当前所在目录的绝对路径

2 查看文件列表

ls /path/显示该目录所有文件或文件夹名
ls -a　　显示所有文件或文件夹名（包含隐藏的）
ls -l　　按列表显示所有文件或文件夹，缩写成ll

ll -h　　友好的显示文件大小（显示成K，MB，GB）

3 创建和删除文件夹

mkdir app 创建app文件夹
makdir -p test1/test2 级联创建文件夹

rmdir app 删除文件夹

4文件操作

rm a.txt 删除文件，需要确认
rm -f a.txt 不询问，直接删除文件
rm -rf 不询问直接删除当前目录下所有文件
rm -rf / no作no die,why you try!

cp 1.txt 2.txt 将1.txt复制为b.txt文件
cp a.txt ../ 将a.txt文件负责到上一层目录文件中

mv a.txt ../ 将a.txt文件移动到上一层目录中

touch 1.txt 创建一个空的文件夹

echo “hello”>1.txt 把“>“左边的输出内容放到右边的文件里去，如果存在就覆盖

vi 1.txt 用vi编辑文本

5 文件打包归档和压缩

tar -cvf test.tar path1 path2 将path1和path2文件夹在当前目录下打包并压缩为test.tar
tar -xzvf file.tar.gz 在当前目录下解压文件
tar -xzvf file.tar.gz -C ../解压到上一层目录中

zip test.txt.zip test.txt 打包并压缩
unzip text.txt.zip 解压

tar常用参数:
-c: 创建一个新的tar文件
-v: 显示运行过程的信息
-f: 指定文件名
-z: 调用gzip 压缩命令进行压缩
-t：查看压缩文件内容
-x: 解开tar文件

6 文件查找

find ./ -name ‘.txt’ 查找以.txt结尾的文件(遍历当前目录)
find ./ -name ‘in‘ 查找以in开头的文件
find ./ -type f 查找普通文件
find ./ -type l 查找连接文件

7 查看文本内容

cat 1.txt 显示文件所有内容
more 1.txt 可以分页看
less 1.txt 不仅可以分页，还可以方便搜索
head -30 1.txt 查看文件前30行
tail -20 1.txt 查看文件尾部20行

8 其他常用命令

echo $ JAVA_HOME 输出变量值

whoami 查询当前登陆的用户名
mkdir test&&cd test

9 添加用户

useradd user1
passwd 123456

-u 指定组ID（uid）
-g 指定所属的组名（gid）
-G 指定多个组，用逗号“，”分开（Groups）
-c 用户描述（comment）
-d 指定用户目录
-e 失效时间（expire date）

10 删除用户

userdel user1

11 修改用户属性

usermod -l user2 user1 user1->user2
usermod -g root usr1 改为root权限
usermod -G h1,h2 user1 给user1添加两个组h1,h2
usermod -d /home/ user1 更改目录

12 用户及用户组相关配置文件

用户配置信息存放位置：
保存用户信息的文件：/etc/passwd
保存密码的文件：/etc/shadow

passwd文件示例：
user002:x:500:500:user002:/home/user002:/bin/bash
passwd文件各字段含义：
account:password:UID:GID:GECOS:directory:shell

shadow文件示例：
user002:$1$vRug41$UUxYzdP0i6s6wtUPieGDQ/:18617:0:99999:7:::

保存用户组的文件：/etc/group
保存用户组密码的文件：/etc/gshadow（设置组管理员时才有用）

13 其他的用户管理命令

id user002　　查看一个用户的UID和GID
su user001　　切换到user001用户
su - user001　　切换到user001用户，并且将环境也切换到user001用户的环境（推荐使用）
exis　　退出当前shell（会退出当前登录用户）

14 网卡地址配置

ifconfig eth0 查看指定的eth0以太网卡地址信息

ifconfig eth0 ip netmask 255.255.255.0
修改ip地址

vi /etc/sysconfig/network-scripts/ifcfg-eth0 修改配置文件来修改ip地址

15 主机名配置管理

修改主机名
vi /etc/sysconfig/network
修改完成后要重启服务器才能生效

主机名-IP映射，服务器本地映射
服务器网络寻址时默认是现在本地的hosts文件中查找IP映射，通过修改hosts来映射局域网内部的主机名非常方便
实现方法，将局域网内的每一台主机的“hostnamip”写入每一台主机的hosts文件中：
vi /etc/hosts
192.168.2.150 user001-server-01
192.168.2.151 user001-server-02
192.168.2.152 user001-server-03

16 网络服务启动与停止

列出系统所有应用服务状态：
service –status-all
重启网络服务：service network restart
停止httpd：service httpd stop
启动httpd：service httpd start
关闭防火墙服务：service iptables stop
关闭防火墙自动启动：chkconfig iptables off

17 查看网络连接信息

netstat:
-a 显示所有连接和监听端口
-l 只显示监听进程
-t (tcp)仅显示tcp相关选项
-u (udp)仅显示udp相关选项
-n 拒绝显示别名，能显示数字的全部转化成数字。
-p 显示建立相关链接的程序名

18 系统管理

df -h　　查看磁盘空间状态信息
du -sh *　　查看指定目录下所有子目录和文件的汇总大小
free　　查看内存使用状况

top　　查看实时刷新的系统进程信息
ps -ef　　查看系统中当前瞬间的进程信息快照
ps -ef | grep myshell.sh　　搜索myshell进程的信息
kill -9 pid　　杀掉进程（-9 表示强制杀死）

19其他系统管理命令

date “+%Y%m%d”　　按格式显示当前系统时间
date -s “2020-01-01 10:10:10”　　设置系统时间
clear　　清屏幕(只是滚到上面看不到了)
uname 显示系统信息。
uname -a 显示本机详细信息。
chmod u=rwx,g=rx,o=rx a.txt　　（u代表所属用户 g代表所属组的成员用户 o代表其他用户）
