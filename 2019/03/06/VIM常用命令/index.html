vim常用命令

vim 常用命令

1 vim 的模式

正常模式（按Esc或Ctrl+[进入） 左下角显示文件名或为空
插入模式（按i键进入） 左下角显示–INSERT–
可视模式

2 文件命令

vim filename 打开单个文件

vim file1 file2 file3 打开多个文件
:open file 打开多个文件中的某个文件
:bn 切换到下一个文件
:bp 切换到上一个文件
:split file 在新窗口打开文件

远程打开文件
:e ftp://../file

3 查找，替换命令

/txt 查找txt，按n 查找下一个，N查找前一个
?txt 查找txt，反向查找，按n 查找下一个，N查找前一个
vim中有一些特殊字符在查找时需要转义　　.*[]^%/?~$

%s/old/new/g 用old替换new,替换整个文件的所有匹配

4 移动命令

: 行好 跳到指定行好
Ctrl + e 向下滚动一行
Ctrl + y 向上滚动一行
Ctrl + d 向下滚动半屏
Ctrl + u 向上滚动半屏
Ctrl + f 向下滚动一屏
Ctrl + b 向上滚动一屏

5 重做，删除，拷贝

Ctrl +r 重做
: 1,5d 删除1-5行
: 1,$d 删除全部
:1,10 co 15 将1-10行插入到第15行之后

:wq 保存并退出
